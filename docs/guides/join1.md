### 游戏内指令操作

1. **注册账号**
    在游戏内输入：
    ```bash
    /reg 密码 密码
    ```
    注意：`/reg` 后面需要输入两遍密码。
    例如：设置密码为 `qwer123y`，则输入：
    ```bash
    /reg qwer123y qwer123y
    ```

2. **登录账号**
    可以使用 `/login` 或 `/l` 指令登录：
    ```bash
    /login 你的密码
    ```
    或
    ```bash
    /l 你的密码
    ```
    例如：注册时用了 `/reg qwer123y qwer123y`，登录时输入：
    ```bash
    /l qwer123y
    ```

3. **修改密码**
    输入以下指令修改密码：
    ```bash
    /changepassword 旧密码 新密码
    ```
    例如：将密码从 `qwer123y` 改为 `aabbcc123`，输入：
    ```bash
    /changepassword qwer123y aabbcc123
    ```
